Partnership Proposal Overview

CyberGuard Solutions Inc. proposes a comprehensive 4-week cybersecurity awareness workshop series designed to equip PCU-DASMA's Information Technology students with essential cybersecurity knowledge and practical skills.

This workshop series aims to bridge the gap between academic learning and industry requirements by providing hands-on experience in network security, data protection, ethical hacking principles, and current cybersecurity threats and countermeasures.

The program will be conducted by certified cybersecurity professionals from CyberGuard Solutions Inc., ensuring that students receive industry-relevant training that will enhance their career prospects in the rapidly growing cybersecurity field.

Program Structure & Curriculum

Week 1: Fundamentals of Cybersecurity

  • • Introduction to cybersecurity landscape and career paths
  • • Understanding common cyber threats and attack vectors
  • • Basic security principles and best practices
  • • Hands-on lab: Setting up secure environments

Week 2: Network Security

  • • Network security architecture and protocols
  • • Firewall configuration and management
  • • Intrusion detection and prevention systems
  • • Hands-on lab: Network vulnerability assessment

Week 3: Data Protection & Privacy

  • • Encryption techniques and key management
  • • Data classification and handling procedures
  • • Privacy regulations and compliance requirements
  • • Hands-on lab: Implementing data encryption solutions

Week 4: Ethical Hacking & Incident Response

  • • Ethical hacking methodologies and tools
  • • Penetration testing techniques
  • • Incident response procedures and forensics
  • • Capstone project: Comprehensive security assessment

Learning Outcomes & Student Benefits

Technical Skills

  • Vulnerability assessment tools usage
